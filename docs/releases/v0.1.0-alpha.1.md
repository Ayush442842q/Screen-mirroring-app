# MirrorLink v0.1.0-alpha.1

First alpha release for MirrorLink.

## Artifacts

- `MirrorLink-v0.1.0-alpha.1-debug.apk`
- `android-debug.apk`

## Notes

- This APK is an installable Android app shell for the native sender surface.
- MediaProjection screen capture and Android WebRTC sender streaming are planned for the next Android milestone.
- Browser laptop sender, browser receiver, and signaling server are already in the repository.

## Verification

- Android debug APK build passed with `:apps:android:assembleDebug`.
