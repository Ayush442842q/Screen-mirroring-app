# MirrorLink Roadmap

This file gives a short public roadmap. The detailed plan lives in [docs/roadmap.md](docs/roadmap.md).

## v1

- Laptop browser sender
- Android mobile sender app
- Browser receiver for mobile, tablet, laptop, and TV browsers
- Room code and QR pairing
- WebRTC streaming
- GitHub Releases with Android APK

## v2

- Better reliability
- Reconnect handling
- Quality controls
- Audio support where possible
- PIN-protected rooms
- Improved TV browser receiver

## v3

- iPhone/iPad sender app
- ReplayKit Broadcast Extension

## v4

- Android TV support
- Remote-friendly TV receiver UI
- TV packaging and release artifacts

## v5

- Production cloud/self-hosting features
- TURN relay documentation
- Stronger security controls
- Optional hosted deployment path
